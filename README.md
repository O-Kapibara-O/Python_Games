# Python Games Collection

## Description:

A set of three simple Python games including: Snake, Tic Tac Toe, Space Invaders.
This project was created for education purpose basic Python programming, using Tkinter .
The code may not follow best practices or modern standards. It was made to practice programming skills.

## How to run:
1. Clone the repository:<br>
  `git clone <repo_url>`
2. Navigate to the project folder:<br>
  `cd python_games`
3. Run the main script:<br>
  `python games.py`


## Project Structure:
```
Python-Games/<br>
│<br>
├── games.py                 # Main script to start the games
├── menu.py                  # GUI menu to select the game
├── snake.py                 # Snake game logic
├── tic_tac_toe.py           # Tic Tac Toe game logic
├── space_invader.py         # Space Invaders game logic
└── utils.py                 # Utility functions
```

## Usage:
After running `games.py`, a menu will appear allowing you to choose a game. Select a game by clicking the corresponding button.
Controls for each game:<br>
Snake -> arrow keys<br>
Space Invader -> arrow keys, spacebar<br>
Tic Tak Toe -> Mouse<br>

## Additional Information:
This project was created several years ago as a learning exercise in Python programming.
Some games have simple logic and can be improved or expanded.
